# Kirby + Parcel + Tailwind minimal starter
## How to use
To get you started 5 step procedure.
```
1. git clone https://github.com/floriankarsten/kirby3-parcel-build-kit project
2. cd project
3. npm install
4. cd htdocs
5. composer install
```

Then point your webserver to ```htodcs/public``` folder.
